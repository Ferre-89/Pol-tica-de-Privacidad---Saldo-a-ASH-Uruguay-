Política de Privacidad - Saldo a ASH (Uruguay)

Información General: Esta aplicación tiene como único objetivo facilitar las donaciones a la ONG "Animales Sin Hogar" a través del sistema oficial de SMS de Antel Uruguay.

Permisos utilizados: La aplicación solicita el permiso SEND_SMS. Este permiso se utiliza exclusivamente para enviar mensajes de texto con la palabra "amigos" al número 2422, según la cantidad definida y confirmada por el usuario.

Uso de Datos: "Saldo a ASH" no recopila, no almacena, no lee ni transmite ningún tipo de dato personal, ni información sobre los mensajes recibidos, contactos o ubicación del usuario.

Transparencia: El proceso de envío es iniciado y confirmado siempre por el usuario. La aplicación no realiza envíos en segundo plano ni de forma oculta.

Terceros: La aplicación utiliza Firebase Crashlytics únicamente para recibir reportes técnicos en caso de errores (crashes), con el fin de mejorar la estabilidad de la herramienta.
